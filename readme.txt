The index of the project is a READ operation on the DB.
The create.php file is a CREATE operation on the DB.
The delete button is a DELETE operation on the DB.
The edit button is an UPDATE operation on the DB.
The search bar is a FILTER operation on the DB.

The 'better-code' folder contains the project written in an intermediate format.
The 'good-code' folder contains the project in a way better fromat as a web application. Best development practise can only be achieved 
by using a framework eg Laravel.
